# Materi 2.1: CRUD - Mengelola Data Berita

## ðŸ“ Apa itu CRUD?
CRUD adalah singkatan dari operasi dasar pengolahan data:
- **C**reate (Buat data baru) -> pakai **POST**
- **R**ead (Baca/Tampilkan data) -> pakai **GET**
- **U**pdate (Ubah data) -> pakai **PUT**
- **D**elete (Hapus data) -> pakai **DELETE**

---

## ðŸ’» Kode Lengkap (server.js)

Berikut adalah contoh server yang bisa menambah, melihat, dan menghapus berita:

```javascript
const express = require("express");
const app = express();
app.use(express.json()); // Wajib ada agar bisa baca data JSON

// Penyimpanan sementara (Memory)
let daftarBerita = [
    { id: 1, judul: "Belajar HTML itu Mudah" },
    { id: 2, judul: "Tips CSS Jadi Jago" }
];

// 1. READ: Melihat semua berita
app.get("/berita", (req, res) => {
    res.json(daftarBerita);
});

// 2. CREATE: Menambah berita baru
app.post("/berita", (req, res) => {
    const dataBaru = {
        id: daftarBerita.length + 1,
        judul: req.body.judul
    };
    daftarBerita.push(dataBaru);
    res.send("Berita berhasil ditambahkan!");
});

// 3. UPDATE: Mengubah judul berita
app.put("/berita/:id", (req, res) => {
    const idUbah = req.params.id;
    const judulBaru = req.body.judul;
    
    // Cari berita yang mau diubah
    const index = daftarBerita.findIndex(b => b.id == idUbah);
    
    if (index != -1) {
        daftarBerita[index].judul = judulBaru;
        res.send("Berita ID " + idUbah + " berhasil diupdate!");
    } else {
        res.send("Berita tidak ditemukan!");
    }
});

// 4. DELETE: Menghapus berita berdasarkan ID
app.delete("/berita/:id", (req, res) => {
    const idHapus = req.params.id;
    daftarBerita = daftarBerita.filter(b => b.id != idHapus);
    res.send("Berita ID " + idHapus + " sudah dihapus!");
});

app.listen(3000, () => console.log("Server CRUD jalan di port 3000"));
```
